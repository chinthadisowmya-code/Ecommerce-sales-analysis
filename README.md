# E-commerce Sales Analysis

## Problem Statement
Analyze sales performance and identify:
- Top-performing products
- Loss-making categories
- Regional sales trends

## Tools Used
- Excel

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. KPI Calculation
4. Dashboard Creation

## KPIs
- Total Sales
- Profit Margin
- Sales Growth
- Top 10 Products

## Dashboard Includes
- Sales by Region
- Profit by Category
- Monthly Sales Trend

## Key Insights
- Technology category generated highest profit
- South region had lowest sales
- Some sub-categories showed negative profit

- Cleaned Dataset
- SQL Queries
